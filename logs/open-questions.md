# 未解之问

> 我自己还没想清楚的问题。这些问题本身就是体系的「活力指标」——
> 一个没有未解之问的体系是死的。

## 当前持有的问题

### 关于 Memory 维度（第四代猜测）
- Memory 真的会沿着 reasoning 的路径被内化吗？还是会以另一种形态出现？
- 如果 memory 内化，「Agent 个体性」这个概念还成立吗（不同用户的 agent 是否还能被认为是同一个模型）？

### 关于多 Agent
- 当 reasoning model 越来越强，多 Agent 是否会进一步收缩到只为「并行加速」服务？
- A2A 协议的演进会让多 Agent 变成「分布式 LLM 系统」吗？

### 关于评测
- Process eval 真的能比 outcome eval 更好预测生产表现吗？
- 在 reasoning trace 隐藏的情况下，怎么做 process eval？

### 关于体系本身
- 七维度的划分稳定吗？是否会需要新增维度？
- 「准维度」中的哪个最有可能升级为正式维度？

## 处理流程
- 这些问题不需要急着回答
- 当某个问题在领域里有了新进展时，标记它，然后归档
- 定期（每月）回顾这份清单，删掉已经不再有意义的问题
